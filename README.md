# Group Hangout (temp name)

Project showcase presented by Team Ratatouille for 2025 UNSW Software Development Society's (DevSoc) Training Program. 

A centralised web application for hosting group hangouts and major events with the add-on features of ~~gambling~~ stakes and community engagement. 

## Stack
- React, TypeScript

## Running the code
Download dependencies with:

```npm install``` or ```npm i```

Then run the code with:

```npm run dev```
